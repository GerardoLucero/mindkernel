# Contributing to Mindkernel

Mindkernel is intentionally minimal. Its value is in being simple enough for anyone to clone and use in under 10 minutes — not in covering every possible life situation.

## What is accepted

- **Bug reports** — the protocol, agents, or skills produce wrong or harmful outputs
- **Documentation improvements** — unclear instructions, missing setup steps
- **New area templates** — context files for areas not yet covered (health, relationships, creative projects)
- **New skill templates** — additional slash command workflows following the existing pattern

## What is not accepted

- Changes to the 3-area core structure (work / money / life)
- Changes to the `CLAUDE.md` routing protocol
- Agent files that contain personal context (names, employers, locations, financial data)
- Features that require external services or APIs

## Process

1. Open an issue before writing anything
2. Wait for acknowledgement
3. One PR per addition — no bundles

## Agent and skill rules

Every file must:
- Work for any person in the world, not a specific profile
- Use plain English with no jargon
- Contain no personal information
- Be completable with zero prior knowledge of the system

## Questions

Open an issue with the `question` label.
